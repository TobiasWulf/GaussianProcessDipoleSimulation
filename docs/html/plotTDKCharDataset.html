Explore TDK TAS2141 characterization dataset and plot its content.

Syntax

Figures = plotTDKCharDataset()

Description

Figures = plotTDKCharDataset() explores the dataset and plot its content in three docked figure windows. Loads dataset location from config.mat.

Examples

figs = plotTDKCharDataset();

Input Arguments

None

Output Arguments

Figure struct which contains the created figure handles.

Requirements

  • Other m-files required: newConfigFigure.m
  • Subfunctions: reconstructMagStimulus, findMinMax
  • MAT-files required: data/TDK_TAS2141_Characterization_2019-07-24.mat, data/config.mat
