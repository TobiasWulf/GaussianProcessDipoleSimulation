Nutzung. Die Anforderungen sind für \gls{ac:cpu}, \gls{ac:ram}, \gls{ac:hdd} näher aufgeschlüsselt. Die Programmierarbeiten mit MATLAB sind
jeweils mit Windows und Linux geschrieben bzw. getestet worden.
